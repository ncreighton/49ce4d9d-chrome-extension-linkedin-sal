# Chrome Extension: LinkedIn Sales Intelligence and Lead Enrichment Tool

A Chrome extension for sales development reps and account executives enriching LinkedIn profiles with buyer intelligence data in real-time. Shows estimated company revenue, technology stack from BuiltWith data, recent funding events, hiring signals, and job change alerts directly on LinkedIn profiles and company pages. One-click export to HubSpot, Salesforce, or CSV, automatic sequence enrollment suggestions based on persona matching, and email and phone number enrichment for contacts.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
